☁️ GitHub Sync

Setelah diisi, tombol ☁️ Simpan ke GitHub akan otomatis push data.js dan file foto ke repo kamu — display akan reload otomatis dalam 30 detik.

Token disimpan di localStorage browser, tidak dikirim ke manapun selain GitHub.

Cara setup (sekali saja):
1. Buat repo di GitHub (public/private bebas)
2. Upload display.html + admin.html ke repo secara manual
3. Aktifkan GitHub Pages → Settings → Pages → Branch: main, folder: / (root)
4. Isi form di atas → klik Test Koneksi → mulai pakai tombol ☁️ di bawah
